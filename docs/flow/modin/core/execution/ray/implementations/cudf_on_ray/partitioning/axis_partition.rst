cuDFOnRayDataframeAxisPartition
"""""""""""""""""""""""""""""""

The class is a base class for any axis partition class based on Ray engine and cuDF storage format. This class provides
the API to perform operations on an axis partition, using Ray as the execution engine. The axis partition is
made up of list of block partitions that are stored in this class.
